# 💊 ASS_PROJECT

Πληροφοριακό Σύστημα Διαχείρισης Φαρμακαποθήκης που αναπτύχθηκε στο πλαίσιο του μαθήματος **Ανάλυση και Σχεδίαση Συστημάτων**.

## ✨Λειτουργίες

* Διαχείριση προϊόντων και αποθέματος
* Διαχείριση πελατών και φαρμακείων
* Καταχώριση πωλήσεων χονδρικής και λιανικής
* Δημιουργία και διαχείριση παραγγελιών
* Έκδοση παραστατικών
* Παραλαβή παραγγελιών προμηθευτή
* Αυτόματη αναπλήρωση αποθέματος όταν πέφτει κάτω από το όριο ασφαλείας

## 👥 Ρόλοι Χρηστών

* Υπάλληλος Αποθήκης
* Πωλητής
* Ταμίας
* Υπάλληλος Εξυπηρέτησης Πελατών

## 🛠️ Τεχνολογίες

* Java (jdk-22)
* Eclipse IDE
* Git & GitHub


## ▶️ Εκτέλεση

Εκτελέστε την κλάση `Main.java` για να ξεκινήσει η εφαρμογή.
